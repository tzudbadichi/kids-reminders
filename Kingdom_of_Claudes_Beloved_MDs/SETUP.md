# SETUP - הקמה והרצה

## דרישות מקדימות
- חשבון Supabase (חינמי).
- דפדפן מודרני.
- שרת מקומי פשוט להגשת קבצים (בגלל ES modules אי אפשר לפתוח קובץ ישירות).

## שלבי הקמה

1. **פרויקט Supabase** — צור פרויקט חדש ב-supabase.com.
2. **סכמה** — ב-SQL Editor הרץ את `supabase/schema.sql`. זה יוצר את הטבלאות, ה-RLS, וטריגר יצירת הפרופיל.
3. **הגדרות התחברות** — Authentication -> Providers: ודא ש-Email מופעל. **חובה לכבות את "Confirm email"** (Authentication -> Sign In / Providers / Settings). ההתחברות באפליקציה היא בשם משתמש + סיסמה, והאפליקציה בונה email פנימי מהשם (`<username>@kids-reminders.app`). בלי כיבוי האישור ההרשמה לא נכנסת מיד וזיהוי "שם תפוס" לא עובד.
4. **קונפיגורציה** — העתק `src/config.example.js` ל-`src/config.js` ומלא:
   - `SUPABASE_URL` ו-`SUPABASE_ANON_KEY` מתוך Project Settings -> API.
5. **הרצה מקומית** — הגש את הקבצים משרת מקומי, לדוגמה:
   - `python -m http.server 5173` ואז `http://localhost:5173/src/`
   - או Live Server ב-VS Code על `src/index.html`.

## בדיקה ראשונה
1. נרשמים עם שם משתמש (אנגלית/מספרים) וסיסמה. אם השם פנוי נכנסים מיד; אם תפוס מתבקשים לבחור אחר.
2. בלשונית "ילדים" מוסיפים שם או שניים.
3. בלשונית "הוספה" בוחרים ילד, תאריך, ומזינים פריטים ידנית (אפשר לדלג על ה-AI). שומרים.
4. בלשונית "היום" בוחרים את התאריך ורואים את הפריטים מקובצים לפי ילד.
